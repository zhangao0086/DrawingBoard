# DrawingBoard
#### <a href="http://blog.csdn.net/zhangao0086/article/details/43836789" target="_blank">博客地址</a>

The drawing board supports:  

1. Pencil
1. Line
1. Dashline
1. Rectangle
1. Ellipse
1. Eraser
1. Custom background
1. Custom brush
1. Undo/Redo
1. Memory-Optimized
